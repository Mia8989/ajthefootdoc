# AJ The Foot Doc

Official website of **Dr. Alton R. Johnson Jr., DPM, DABPM, FACPM, FASPS, FFPM RCPS (Glasg), CWSP**: board-certified podiatric surgeon, Certified Wound Specialist Physician, and Clinical Assistant Professor at the University of Michigan Medical School.

Static site (HTML, CSS, vanilla JS). No build step. Hosted on GitHub Pages; intended custom domain: **ajthefootdoc.com**.

## Structure
- `index.html`: home
- `about/`, `media/`, `blog/` (plus one post), `resources/`, `contact/`
- `assets/`: `css/`, `js/`, `images/` (WebP), `downloads/` (patient and student PDFs)
- `sitemap.xml`, `robots.txt`, `llms.txt`: SEO and AI-crawler layer
- `CNAME`: custom domain for GitHub Pages

Motion: Lenis smooth scroll plus GSAP / ScrollTrigger, loaded from CDN.
